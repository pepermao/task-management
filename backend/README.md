<h1 align="center">Task Management App Server Side</h1>

## Installation
- Access Folder
  ``` sh
  cd backend
  ```
- Startup Mongo DB via Docker:
  ``` sh
  docker-compose up -d
  ```
- Install packages:
  ``` sh
  npm install
  ```
- To run in dev mode
  ``` sh
  npm run start
  ```
- Access http://localhost:3000 in your browser.
